## 1. 산삼 펫이란

"산삼 펫" 사이트는 산삼을 전문적으로 키우는 분들을 위한 온라인 플랫폼입니다🌱  
이 사이트는 전문 심마니들이 유저들의 소중한 산삼을 훌륭하게 키울 수 있도록 도와줍니다.  
산삼 숙성 장인들에게 맡기고 싶은 유저들을 모집합니다🎈
